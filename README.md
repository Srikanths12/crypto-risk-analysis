
# Cryptocurrency Market Risk Analysis (Java + Machine Learning)

This project analyzes cryptocurrency market risk using basic machine learning techniques in Java.

## Features
- Cryptocurrency data analysis
- Risk prediction model
- Console-based analysis output

## Technologies Used
- Java
- Machine Learning (Weka)
- Maven

## Project Structure
crypto-risk-analysis
│
├── pom.xml
├── README.md
└── src
    └── main
        └── java
            └── com
                └── crypto
                    ├── CryptoRiskApplication.java
                    └── RiskAnalyzer.java

## How to Run
mvn compile
mvn exec:java
